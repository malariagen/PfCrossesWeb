'Find Gene' wizard

This wizard searches for genes of interest in the [@organismshort] genome in two ways:
  • By name or description
    Enter a keyword (or partial keyword). The wizard will retrieve a list of all genes having this keyword (or partial keyword) in the name and description fields. Select a specific gene from this list and click 'Finish' to open a pop-up with more information on the selected gene.
  • By genomic region
    Select a chromosome from the drop-down list, and specify a start and stop position in base pairs. The wizard will display a list of all genes within this region. Select a specific gene from this list and click 'Finish' to open a pop-up with more information on the selected gene.

Recent hits

The 'Recent hits' section beneath the search options displays links to genes that were found with this wizard earlier in the active session. Click on a gene name to jump directly to results for that gene.

Gene info pop-ups

Selecting a specific gene will open a gene info pop-up with basic information about this gene:
  • Names
  • Description
  • Position on the [@organismshort] genome
A set of buttons leads to different views, each visualising data about the gene:
  • List of variants in the gene
  • The position on the genome
  • Any public genotypes for the gene
  • View the gene in PlasmoDb
